# OKR Framework Template — Google Style
### Performance Management · Monitoring Kinerja Kuartalan

---

## 📋 Cara Menggunakan Template Ini

1. Tentukan **3 Objectives** utama per kuartal (maksimal 5)
2. Setiap Objective memiliki **3 Key Results** yang terukur
3. Input **Realisasi** di akhir periode evaluasi
4. Sistem otomatis menentukan **Traffic Light Status**
5. Susun **Development Plan** berdasarkan hasil

## 📊 Ringkasan Performa

```
┌─────────────────────────────────────────────────────┐
│  OVERALL OKR SCORE                                  │
│                                                     │
│  Score OKR 1   :  _____%                           │
│  Score OKR 2   :  _____%                           │
│  Score OKR 3   :  _____%                           │
│                                                     │
│  OVERALL SCORE :  _____%                           │
│  STATUS        :  🟢 On Track / 🟡 At Risk / 🔴 Off │
└─────────────────────────────────────────────────────┘
```

| Range Score | Rating | Keterangan |
|-------------|--------|-----------|
| 90–100% | ⭐ Exceptional | Jauh melampaui target |
| 75–89% | ✅ Exceeds Expectation | Melampaui target |
| 60–74% | ✔️ Meet Expectation | Sesuai target |
| 40–59% | ⚠️ Below Expectation | Di bawah target |
| < 40% | ❌ Unsatisfactory | Jauh di bawah target |

---

## 🚦 Traffic Light Status

| Status | Syarat | Tindakan Atasan |
|--------|--------|----------------|
| 🟢 **On Track** | Score ≥ 70% | Apresiasi & dorong untuk maintain |
| 🟡 **At Risk** | Score 40–69% | Coaching session & review hambatan |
| 🔴 **Off Track** | Score < 40% | Action plan wajib + eskalasi ke atasan |

## 📅 OKR Timeline Recommended

```
AWAL KUARTAL (Minggu 1–2)
└─ Set OKR bersama atasan, pastikan semua KR SMART

MID-QUARTER CHECK-IN (Bulan 2)
└─ Review progres, identifikasi hambatan, update status

AKHIR KUARTAL
├─ Isi realisasi semua KR
├─ Hitung score & tentukan status Traffic Light
├─ Coaching conversation dengan atasan
└─ Susun OKR kuartal berikutnya
```
