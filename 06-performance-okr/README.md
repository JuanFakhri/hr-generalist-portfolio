## 📌 Problem Statement

Perusahaan belum memiliki sistem monitoring kinerja yang terstandarisasi. Target per divisi tercampur tanpa kerangka evaluasi yang jelas — menyulitkan pengukuran progres, pemberian feedback, dan coaching yang tepat sasaran.

---

## 🎯 Tujuan & Output

| Tujuan | Output yang Dihasilkan |
|--------|----------------------|
| Standarisasi monitoring kinerja | Google-Style OKR Performance Review Template |
| Visualisasi progres yang jelas | Sistem Traffic Light scoring per Key Result |
| Panduan coaching berbasis data | Development Plan per karyawan |

---

## 📊 Ringkasan Hasil

```
Jumlah Objectives       : 3
Total Key Results (KR)  : 9 KR  (3 KR per Objective)
Sistem Scoring          : Traffic Light (On Track / At Risk / Off Track)
Periode Evaluasi        : Kuartalan
Format                  : Google Sheets — siap pakai & mudah dimodifikasi
```

## 🚦 Sistem Traffic Light Scoring

| Status | Kondisi | Tindakan |
|--------|---------|---------|
| 🟢 **On Track** | Progress ≥ 70% dari target | Pertahankan — lanjut sesuai rencana |
| 🟡 **At Risk** | Progress 40–69% dari target | Perlu perhatian — review & coaching |
| 🔴 **Off Track** | Progress < 40% dari target | Eskalasi — action plan segera |

---

## 🔢 Formula Scoring

```
Score per KR  = (Realisasi / Target) × 100%

Score per OKR = Rata-rata Score seluruh KR dalam 1 Objective

Overall Score = Rata-rata Score seluruh Objectives
             = (Score OKR1 + Score OKR2 + Score OKR3) / 3
```
