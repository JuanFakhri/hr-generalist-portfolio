Folder ini berisi **template dokumen HR** yang dikembangkan selama HR Bootcamp Batch 22 dan dapat langsung digunakan atau diadaptasi sesuai kebutuhan perusahaan.

---

## 🎯 Cara Menggunakan

1. **Pilih template** yang sesuai dengan kebutuhan
2. **Salin konten** ke Google Docs / Microsoft Word
3. **Sesuaikan** nama perusahaan, posisi, dan kriteria
4. **Gunakan** sebagai dokumen resmi proses rekrutmen
